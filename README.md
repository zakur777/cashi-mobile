# cashi-mobile

Scaffold inicial de Cashi Mobile con **Expo + TypeScript + Expo Router**.

## Requisitos

- Node.js 20+
- npm 10+

## Comandos base

- `npm install`
- `npm start`
- `npm test`

## Stack de Unit 1

- React Native + Expo
- Expo Router (file-based routing)
- AsyncStorage
- Zod
- Jest + jest-expo
